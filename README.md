## Oped Document Generator library
  Allows to create and store an ODT file. Supports:
  - styles
  - paragraphs
  - headers
  - tables
  - tables of content

## Sample usage
  ```PHP
  $document = new ODTDocument();
  Sdocument->create();
  ```
